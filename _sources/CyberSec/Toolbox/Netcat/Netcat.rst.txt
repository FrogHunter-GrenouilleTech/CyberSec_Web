####

Netcat is a command line tool allowing to read and write data through the network. For the Data
transfert, Netcat is using TCP and UDP.

Because of its versatility it is also know of "The Swiss Army Knife of the TCP/IP". It can be used
both as a server or as a client.

-----------------------
some use case of Netcat
-----------------------

    * Diagnose errors and problems endangering the network.
    * Port Scanning
    * Data Streaming
    * Simple data transfers.
    * Configure chat and web servers
    * Launching requests by Email.
